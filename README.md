# Lancet & Linen / Lanzett & Leinen

Full-featured medieval **life & business simulation** about a German **Bader** (barber-surgeon) in the Holy Roman Empire (c. 1382).

Inspired by *The Guild* (property, staff, politics, household) and itinerant barber-surgeon life — **original title and systems**.

## Play

```bash
npm install
npm run dev
```

Open **http://localhost:5173/**  
Mobile: landscape recommended; PWA manifest for “Add to Home Screen”.

## Feature set

### Core loop
- Treat patients (humors, techniques, skill check)
- Run stall / **Badestube**, daily costs, market, crafting
- Travel Franconia by horse cart + road events
- City mentors for techniques
- Story campaign Acts I–III + multiple endings (incl. dynasty)

### Guild-like systems
- **Multi-city property** (stall, bath, home, warehouse) + managers
- **Staff** with loyalty, wages, gifts, training, quit risk
- **Family** courtship, marriage, heir
- **Politics** offices, titles, bribes, church alms
- **Prestige** & fee multipliers from titles
- **Journal** chronicle + quest log
- **City events** & seasonal festivals
- **Tutorial tips** + settings (gore, text, tips, effects)

### Meta
- Manual **Save / Load / Export / Import**
- Auto-save on major actions
- EN + DE
- Procedural audio
- Mobile touch + scale

## Stack

Vite · TypeScript · Phaser 4 · Web Audio · i18next
